Project 1: Datastructure and Algorithm Simulator



Type :

Desktop Application

started :

1 April 2019

completed :

1 August 2019

This application was developed for my research in my final year of my Computer Science Degree. It is a desktop application i created using Java only. It is intended to be used by users who want to get better understanding of various datastructures and sorting algorithms, the software shows how various datastructures and Algorithms work. Users use this program to experiment with various datastructure such as BST trees, Red-Black Trees etc, aswell as sorting algorithms such as Insertion sort, Selection sort etc. I love this project because it tested my programming skills and it was a project i invested alot of time in.
